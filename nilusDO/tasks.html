🔥 Connect Firebase

One-time setup to sync tasks between your PC and iPad in real time.

1
Go to console.firebase.google.com → Create a project → Disable Analytics → Continue
2
Build → Realtime Database → Create database → Start in test mode
3
Project Settings (gear) → Your Apps → </> Web → Register app → Copy the config
4
Open tasks.html in a text editor and replace the FIREBASE_CONFIG at the top with your config
const FIREBASE_CONFIG = {
  apiKey: "AIza...",
  databaseURL: "https://your-project-rtdb.firebaseio.com",
  ...
};
